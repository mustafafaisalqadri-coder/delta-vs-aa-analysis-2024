# Delta vs American Airlines - Operational Efficiency Analysis 2024

## What This Project Does
Analysis of 2024 US flight data comparing Delta Airlines
and American Airlines operational performance across
900,000+ flights using real BTS government data.

## Key Questions Answered
- Which airline has fewer controllable delays?
- Which months are worst for each airline?
- Which hubs perform best and worst?
- What percentage of delays can each airline actually control?

## Tools Used
- Python (pandas, matplotlib)
- Data Source: US Bureau of Transportation Statistics
- Visualization: matplotlib

## Files
- combine_data.py: Downloads and combines 12 months of data
- compare_AA_vs_DL.py: Full comparison analysis and charts
